# SQL-injection-course

Интерактивный учебный тренажёр для изучения SQL-инъекций. 
Курсовой проект студента КЗИ-252 Автушенко М.С.

##  О проекте
Три уровня сложности:
- **Error-based SQLi** - базовый уровень с выводом ошибок
- **Union-based SQLi** - средний уровень с UNION-инъекциями
- **Blind SQLi** - продвинутый уровень (логические/временные инъекции)
